Solution 4.2.1.1.

  1. We have \(P_2 = P_1 + 0.1\cdot P_1 = 1.1\cdot P_1.\) Similarly, \(P_3 = P_2 + 0.1\cdot P_2 = 1.1\cdot P_2\text{.}\)

  2. We may reasonably extrapolate that \(P_t = 1.1\cdot P_{t-1}\text{.}\)

  3. We see \(P_2 = 1.1 P_1 = 1.1 (1.1 P_0) = (1.1)^2 P_0\text{.}\) Similarly, \(P_3 = (1.1)^3 P_0\text{.}\) These formulas are more useful because we only need to know the first term, \(P_0\text{.}\)

  4. Best guess: \(P_t = (1.1)^t P_0\text{.}\) Much like the linear model from the previous activity, this model assumes that the population continues to grow forever.

  5. You should notice that the curve grows more slowly if \(r \lt 0.1\text{,}\) and more quickly if \(r \gt 0.1\text{.}\)
